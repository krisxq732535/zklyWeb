我们的观点

智能制造实现之路：“三化融合”

精益化为思想、信息化为神经网络、自动化为四肢，构建出智能工厂有机体，为企业在智能制造转型中实现价值的落地

装备制造如何实现拉动式生产

“拉动”式生产是以准时管理为核心的精益制造体系的中心思想之一。真正实现拉式生产，需要整个企业内部上下游，甚至供应链整体水平的同步提升。在生产线上看任何一道工序，都不能将其孤立在生产线之外。
